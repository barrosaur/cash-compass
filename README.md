# CashCompass

**CashCompass** is a mobile expense tracking application designed to help users record, organize, and monitor their daily financial activities. The application allows users to manually track expenses and income, categorize transactions, store receipts, and visualize spending patterns through graphs and organized records.

The goal of CashCompass is to promote better financial awareness and budgeting habits by providing users with a convenient tool to track how their money is spent over time.

---

# Project Overview

CashCompass is designed primarily for individuals who want a simple and organized way to manage their personal finances. This includes:

- Students managing daily or weekly allowances
- Individuals tracking personal spending
- Users who want to monitor financial habits

The system helps users:

- Record financial transactions
- Track expenses and income
- Categorize spending
- Set budget goals
- Review financial records

By making financial activity easier to monitor, CashCompass encourages users to develop better budgeting and spending habits.

---

# Features

The application includes the following core features:

### Transaction Management

- Record financial transactions including:
  - Expenses
  - Income
  - Transfers
- Attach optional receipt images
- Store and organize transaction history

### Expense Categorization

- Use predefined categories
- Create custom categories
- Organize expenses by type (e.g., food, transportation, school supplies)

### Account Management

Support for multiple account types including:

- Cash on Hand
- Bank Account
- Credit Card
- GCash
- Custom Accounts

### Budget Goals

- Create personal budget goals
- Define target amounts
- Manually track whether goals are achieved

### Expense Tracking Views

Users can view their financial activity by:

- Daily records
- Weekly records
- Monthly records

### Reminder Notifications

- Daily reminder notifications (5:00 PM) to log expenses.

### Built-in Calculator

- Helps users quickly perform calculations when entering transactions.

### Receipt Storage

- Users may optionally capture or upload receipt images.

# Application Screens

The application includes the following primary screens:

- **Records Screen** – Displays the user's transaction history.
- **Transactions Screen** – Allows users to record income, expenses, or transfers.
- **Accounts Screen** – Manage financial accounts.
- **Categories Screen** – Manage expense categories.
- **Budget Goal Screen** – Create and track financial goals.
- **Settings Screen** – Configure application preferences.

# System Requirements

The application is intended to run on modern smartphones that support:

- Android or iOS
- Camera access (for receipt capture)
- Local device storage
- Push notifications

---

# Performance Requirements

The system is designed to meet the following performance targets:

- Application launch and records display within **1–2 seconds**
- Transaction saving within **1 second**
- Support for at least **5,000 stored transactions**
- Reminder notifications delivered near **5:00 PM daily**

---

# Security and Privacy

CashCompass prioritizes user data privacy.

Key security considerations include:

- Financial data stored **locally on the device**
- No automatic transmission of sensitive financial data
- Optional device-level authentication (PIN, fingerprint, Face ID)
- Secure storage of transaction data
- Notifications that avoid displaying sensitive financial information
